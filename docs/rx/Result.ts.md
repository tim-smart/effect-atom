### With (type alias)

**Signature**

```ts
export type With<R extends Result<any, any>, E, A> = R extends Initial<infer _E, infer _A>
  ? Initial<E, A>
  : R extends Success<infer _E, infer _A>
  ? Success<E, A>
  : R extends Failure<infer _E, infer _A>
  ? Failure<E, A>
  : never
```

Added in v1.0.0
